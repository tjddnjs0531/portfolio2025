View

강서구문화통합시설 홈페이지 구축 프로젝트에 참여하여, 강서구 내 주요 문화시설을 하나의 플랫폼에서 통합 관리·운영할 수 있도록 개발하였습니다. 본 사이트는 강서아트리움, 허준박물관, 겸재정선미술관, 강서문화원 등 4개의 개별 기관을 아우르며, 각 시설의 특성을 반영한 콘텐츠 구성과 일관된 사용자 경험 제공에 중점을 두었습니다.

특히, 부평구문화재단 프로젝트와 유사한 구조를 가지면서도 전체 프로젝트 기간이 기존 대비 약 50% 단축된 상황에서 진행되어, 한정된 일정 속에서도 퍼블리싱 작업의 효율성을 극대화하는 데 집중하였습니다. 이를 위해 재사용 가능한 컴포넌트와 모듈화된 구조를 적극 활용하여 개발 시간을 단축하고 유지보수의 용이성을 확보하였습니다.

또한, 복잡한 정보 구조를 체계적으로 정리하고, 사용자가 원하는 문화·예술 관련 정보를 빠르게 찾을 수 있도록 UI/UX 최적화를 진행하였으며, 웹 접근성과 반응형 디자인을 적용하여 다양한 디바이스에서 원활한 이용이 가능하도록 구현하였습니다.
