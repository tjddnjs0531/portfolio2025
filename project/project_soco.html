View

공동체주택 홈페이지 구축 프로젝트는 첫 번째 프로젝트로, 새로운 환경에서 다양한 도전을 경험한 의미 있는 작업이었습니다.

본 프로젝트에서는 공동체주택, 사회주택, 청년주택 3개 패밀리 사이트를 모두 구축해야 했으며, 일정 문제로 인해 디자인이 완전히 나오지 않은 상태에서 기획서와 기존 페이지를 참고해 작업을 진행해야 하는 경우가 많았습니다. 이를 극복하기 위해 일관된 디자인 시스템을 유지하면서도, 각 사이트의 특성을 살릴 수 있도록 유연하게 퍼블리싱을 적용하였습니다.

특히, 주택찾기 시스템 내 주택 정보 페이지는 다양한 유형의 주택 정보를 가독성 있게 정리하는 것이 핵심 과제였습니다. 이를 위해 유동적인 콘텐츠 구조를 적용하여 정보가 효율적으로 배치될 수 있도록 구성하였으며, 가독성과 사용자 경험을 고려한 UI 설계를 통해 직관적인 정보를 제공하도록 구현하였습니다.

퍼블리셔로서 제한된 환경 속에서도 디자인 해석 능력과 문제 해결 능력을 키울 수 있었던 프로젝트였으며, 구조적인 코드 작성과 유지보수의 편의성을 고려한 퍼블리싱 작업을 수행하였습니다.
