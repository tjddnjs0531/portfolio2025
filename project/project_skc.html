Detail

View

손기정기념관 홈페이지 구축 프로젝트에 참여하여, 손기정 선생의 업적과 역사적 가치를 효과적으로 전달할 수 있도록 웹사이트를 개발하였습니다.

웹 접근성을 고려한 시멘틱 마크업과 최적화된 스타일링을 적용하여, 다양한 연령대의 사용자가 편리하게 이용할 수 있도록 구축하였습니다. 또한, 반응형 웹 구현을 통해 모바일·태블릿·PC 등 다양한 디바이스에서 최적의 사용자 경험을 제공하도록 설계하였습니다.

퍼블리셔로서 일관된 UI 유지, 성능 최적화, 유지보수 편의성을 중점으로 작업하였으며, 효율적인 코드 구조를 적용하여 향후 확장성과 운영의 용이성을 높였습니다.
